Проект представляет из себя Spring Boot Web MVC приложение для управления библиотекой с 3 таблицами БД: авторы, книги и комментарии
Эти таблицы связаны между собой связью "Один ко многим".
Были реализованы 3 класса с сущностями этих таблиц с валидацией данных; для управления сущностями в системе 3 контроллера, которые обрабатывают
запросы, связанные с операциями для сущностей. Далее 3 репозитория для работы с сущностью и БД, доступ к данным; 3 сервиса для бизнес-логики, 
взаимодействие между контроллером и репозиторием. А также файлы .html для дружественного интерфейса.
# Функциональность
## 1) Просмотр всех авторов, книг и комментариев
## 2) Добавление новых авторов, книг и комментариев
## 3) Удаление авторов, книг и комментариев
## 4) Редактирование существующих авторов, книг и комментариев
## 5) Валидация вводимых данных
